# KFBot

此仓库代码配合蓝牙HID鼠标模块，应用于Kirara Fantasia (きらファン) 手游的自动挂机。

## 所需硬件参考

1. Arduino Uno；
2. BTKM-05（DEMO板，内含的蓝牙鼠标模块星号为XM-04-HID-M）；
3. 10kΩ电阻2个、20kΩ电阻1个、220Ω电阻一个；
4. 导线若干；
5. LCD1602液晶屏幕、电位器2个。

## 开发用测试机型

iPhone 6s (iOS 13.4)
